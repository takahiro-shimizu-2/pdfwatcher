パフォーマンステスト手順

  1. Chrome DevToolsを開く (F12)
  2. Performanceタブで記録開始
  3. PDFWatcher拡張機能を実行
  4. 記録を停止し、処理時間を確認
  5. Memory タブでヒープスナップショットを取得
